ViSi Remote

ViSi is an iPad app which is used to control Soundcraft's Mixers remotely. There was an existing app where only the Faders were functioning. I was the only UI Designer for this project and my responsibility in this project was to redesign the look and feel of this app and design screens for more functionality added like Processing Sound Quality, Dynamics, Metering Screens etc were to be added.

This App was required to look exactly like a real Mixing console and the main task was to create a Skeuomorphic UI for real looking controls. I worked with 2 Product Managers to understand the actual Use Case of the new functionality added and tried to prepare best possible Interface for these.
