Confer Control

Confer Control is a Windows PC based Application to control AKG's conferencing units CS5 and CS3. This software has multiple workflows to manage conferences which includes, Seating, Venue Creation, Reporting, Projection Etc.

I worked with a Product Manager for this project who provided wireframes for all workflows which were made in Balsamiq. I prepared a few themes for initial proposals and finally Black, Ochre Yellow and White theme was approved by all stakeholders. All screens and the UI guide were prepared using Illustrator."
